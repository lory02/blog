html（一）

发表于 2019-11-19 | 分类于 html

html（全称 HyperText Markup Language ）,是一种超文本标记语言，html使用标记标签来描述网页，它包含了html标签及文本内容

html基本结构

  • < !DOCTYPE html > 声明为 HTML5 文档（用于区分不同版本的html）

  • html根元素

  • head元素包含文档的元数据

  • body元素包含了可见的页面内容

  • icon ：显示网页图标

    • 如果没有插入link，默认浏览器会读取名为favicon.ico的文件（有时候没有设置图标浏览器会在控制台报错）

  • title 标题元素

    • 如果不出现在head内，将自动移到head内
    • 出现多个的话，仅第一个生效

  • 在 vscode 中输入感叹号，可以自动生成html模板

html标签

头部标签

  • meta ：用来描述HTML文档的描述，关键词，作者，字符集等。 

    • <!-- 设置字符集：用于中文编码 -->
<meta charset="UTF-8"> 
<!-- 适配 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta http-equiv="X-UA-Compatible" content="ie=edge">

  • script ： src引入js文件

  • style：在style标签中写css

  • title ： 文档标题

  • base：基本的链接地址

    • <head>
    <!-- 设置文档的基准地址 -->
    <base href="https://www.baidu.com/images/" target="_blank">
</head>

<body>
    <!-- 完整地址: base地址与img地址的拼接 "https://www.baidu.com/images/logo.png"-->
    <img src="logo.png"> 
</body>

  • autdio

  • progress ： 进度条 （可以通过改变value值来，来显示加载情况）

    • <progress value="50" max="100"></progress>

<p>a&nbsp;b</p>
