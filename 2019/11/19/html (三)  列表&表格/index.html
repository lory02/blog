html (三) 列表&表格

发表于 2019-11-19 | 分类于 html

列表

  • ol&li

    • ordered list & list item

    • 有序列表，< ol >中只能嵌套< li >,< li >中可以嵌套其他元素

  • ul&li

    • unordered list & list item

    • 无序列表，< ul >中只能嵌套< li >,< li >中可以嵌套其他元素

  • dl&dt&dd

    • desciption list & description term & description description

    • 自定义列表

表格

table 标签定义 HTML 表格。简单的 HTML 表格由 table 元素以及一个或多个 tr、th 或 td 元素组成；由thead（一个tabel只能有一个thead），tbody，tfoot组成表单结构。

标签

  • table ： 表格

  • tr 行

  • th 表头

  • td 表数据

  • colgroup

    • 用于对col标签进行组合，以便对其进行格式化。添加class属性就能用css来设置样式。

属性

  • colspan ： 设置跨列

  • rowspan： 设置跨行

    • 设置了跨行或跨列，相应需要删除多余的表格（td）

  • border-spacing：边框间距

  • cellspacing：边框间距

map & area

  • map : 映射标签

    • usemap的属性值和map元素的“id”或者“name”相等

  • area ： 圈定map中位置

    ​ shape=”rect” 矩形对角点坐标 “ x1，y1，x2，y2 ”

    ​ shape=”circle” 圆坐标 圆心+半径 “ x1，y1，r “

    ​ shape=”poly” 路径坐标 “ x1，y1，x2，y2，x3，y3，xn，yn ”
