html（二）表单

发表于 2019-11-19 | 分类于 html
1.form

用于为用户输入创建 HTML 表单，表单用于向服务器传输数据。

  • form属性：（涉及到http协议的内容，后续再细写）
    • action :action=“URL”, 点击按钮提交表单，向何处提交表单数据。
    • target:规定在何处打开 action URL。
    • method ：get&post，规定用于发送 form-data 的 HTTP 方法。
    • enctype，编码方式。表单数据提交时使用的编码类型，默认使用”pplication/x-www-form-urlencoded”，如果是使用POST请求，则请求头中的content-type指定值就是该值。如果表单中有上传文件，编码类型需要使用”multipart/form-data”，类型，才能完成传递文件数据。
      （content-type用于填写数据格式，服务器根据该信息可以知道如何对数据进行解码）
      • application/x-www-form-urlencoded
        • 数据之间用&相连 key1=val1&key2=val2
      • multipart/form-data
        • 传输文件
      • text/plain
2.input

用于搜集用户信息。根据不同的type属性值，输入字段拥有很多形式。

  • type属性

    • text ： 当行文本输入

    • password： 密码输入框

    • radio : 单选

    • checkbox ： 复选框

    • button : 普通按钮

    • sumbit : 提交按钮

    • reset ： 重置按钮

    • file ： 文件上传 （ accept=“ MIME_type”）

      • accept ： 可接受的文件类型

        •  <input type="file" name="" id="" accept="image/*,text/*">
5.select

可创建单选或多选菜单

  • select元素的value值 = 选中选项的value值 （如果选项没有value属性，默认value为选项的textContent值）

  • option： 用于定义列表选项

  • opgroup： 用于选项分组

  • opgoup的label属性 ： 选项组描述

  • <select name="age">
    <option value="" hidden>请选择</option>
    <optgroup label="少年">
        <option value="1-6">1-6岁</option>
        <option value="6-12">6-12岁</option>
    </optgroup>
    <optgroup label="青年">
        <option value="13-15">13-15岁</option>
        <option value="16-18">16-18岁</option>
        </optgroup>
    <optgroup label="少年">
        <option value="18-22">18-22岁</option>
        <option value="23-30">23-30岁</option>
    </optgroup>
</select>
